Useful OT functions:
1. Add a query param to override geolocation: ?otgeo=ca,qc
2. Check current geolocation: Optanon.getGeolocationData()
3. Resurface consent: Optanon.ToggleInfoDisplay()
4. Change language: OneTrust.changeLanguage('fr') // Looks like OneTrust object is the main handle for OT to do what it does. It contains all kinds of useful methods.
